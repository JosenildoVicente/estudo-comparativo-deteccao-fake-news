# Estudo Comparativo de Detecção de Notícias Falsas
Este repositório contém o código fonte e os recursos utilizados no estudo comparativo de algoritmos de detecção de notícias falsas, com um artigo de nome "Detecção de Fake News: Uma Análise Comparativa". O estudo foi desenvolvido como parte de um projeto acadêmico para analisar a eficácia de diferentes abordagens na detecção de conteúdo enganoso em conjuntos de dados reais.

### Descrição do Projeto
O objetivo principal deste estudo foi investigar e comparar o desempenho de diversos algoritmos de aprendizado de máquina na detecção de notícias falsas em diferentes conjuntos de dados. Foram utilizadas técnicas de pré-processamento de texto, extração de recursos e classificação, com a avaliação do desempenho dos modelos por meio de métricas específicas.

### Estrutura do Repositório
- **/Codigos:** Inclui os scripts Python utilizados para pré-processamento, extração de recursos, classificação e avaliação dos modelos.
- **/matrizConfusao:** Armazena as matrizes de confusao que foram obtidos durante a execução do projeto.
- **/WordClouds:** Contém as núvens de palavras dos conjuntos de dados utilizados.
- **/gridcv:** Armazena os modelos já treinado dos algoritmos de classificação.
- **/predicao:** contém as predições feitas pelo modelo em arquivo csv.

### Como Utilizar
Para reproduzir os experimentos realizados neste estudo, siga estas etapas:

1. Clone este repositório em sua máquina local.
2. Instale as dependências necessárias listadas no arquivo requirements.txt.
3. Execute os scripts na ordem apropriada.
4. Analise os resultados obtidos e faça as adaptações necessárias conforme sua necessidade.
